Для использования автотеста по созданию ЛНД необходимо:

1. Скачать вместе с автотестом файл "Инструкция по работе с КЭДО Lexema" в формате jpeg

2. Переместить данный файл в путь: C:\Users\User

3. Для успешного прохождения автотеста, в файле формата Excel "File" необходимо во вкладке "ЛНД" в ячейке A1 указать подтип ЛНД, используемый на проекте, например, "Политики организации". 
Если подтип не будет изменен, то возможно, в выпадающем списке будет отсутствовать подтип указанный в примере , что приведет к невозможности пройти дальнейший выбор подтипа.
